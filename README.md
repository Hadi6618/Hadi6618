## 🚀 Featured Projects

### 🔹 [PRISM](https://github.com/Hadi6618/PRISM)
Unified VAD framework that fuses:
- **STG-NF** (pose/object-centric stream)
- **MULDE** (frame/appearance-centric stream)

PRISM performs alignment, normalization, smoothing, and weighted late fusion for frame-level anomaly scoring.

### 🔹 [STG-NF](https://github.com/Hadi6618/STG-NF)
Extended STG-NF architecture with attention variants:  
`none`, `skeleton`, `frame`, `dual`, `triplet`.

### 🔹 [MULDE](https://github.com/Hadi6618/MULDE)
Multiscale log-density estimation implementation used as the appearance stream in PRISM.
